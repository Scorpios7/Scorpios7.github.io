idea快捷键

发表于2023-12-11|更新于2023-12-11
|字数总计:157|阅读时长:1分钟|阅读量:

Ctrl+D 复制一行

Ctrl+X 删除一行

SHIFT + ENTER 在该行下面另起一行

Ctrl + Alt + L 格式化代码

Ctrl + Alt + T 生成try catch

Ctrl + G 定位行

双击Shift 查找任何内容

Ctrl + W 选中当前内容

Ctrl + 左方向键：光标跳转到当前单词/中文句的左侧开头位置

Ctrl + 右方向键：光标跳转到当前单词/中文句的右侧开头位置

Ctrl + shift + 上下方向键：上下移动代码行

Ctrl + Alt + B：跳转到方法实现处

多光标 {

​ ALT + SHIFT + 鼠标点击

​ Ctrl + Shift + Alt + J 一步到位

​ ALT + 鼠标下拉

}
